# 🌱 Sistema Usuarios

API REST desenvolvida com Spring Boot para gestão de usuários com autenticação JWT.

---

## 🚀 Tecnologias Utilizadas

- Java 17
- Spring Boot
- Spring Security
- JWT (JSON Web Token)
- PostgreSQL
- Docker
- Docker Compose
- JPA / Hibernate

---

## 🔐 Funcionalidades

- Login com JWT
- Controle de acesso por ROLE (ADMIN / USER)
- CRUD completo de usuários
- Inicialização automática de admin
- Ambiente totalmente containerizado

---
